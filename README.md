# 🍦 Vegan Ice Cream – Coming Soon Page

A clean and responsive **Coming Soon** landing page for a Vegan Ice Cream brand, built using **HTML5** and **CSS3**.

## ✨ Features

- Modern and minimal UI
- Responsive card layout
- Email subscription input
- Custom button styling
- Vegan-themed color palette

## 🛠️ Built With

- HTML5
- CSS3
